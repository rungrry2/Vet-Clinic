✅ 1. เรียกใช้ Scalar Function

👉 fn_CalculateBillTotal

🔧 ใช้แบบ SELECT
SELECT dbo.fn_CalculateBillTotal('B001') AS total_price;

👉 ผลลัพธ์:

total_price
-----------
500.00
💡 ใช้ร่วมกับตารางก็ได้
SELECT bill_id, dbo.fn_CalculateBillTotal(bill_id) AS total
FROM Bills;

👉 อันนี้โคตรโปร (อาจารย์ชอบ)

✅ 2. เรียกใช้ Table-Valued Function

👉 fn_GetPetMedicalHistory

🔧 ใช้เหมือน SELECT จาก table
SELECT *
FROM dbo.fn_GetPetMedicalHistory('P001');
💡 ใช้ร่วมกับ WHERE / ORDER BY ได้
SELECT *
FROM dbo.fn_GetPetMedicalHistory('P001')
ORDER BY visit_date DESC;

👉 ✔ ORDER BY ต้องมาเขียนตรงนี้ (ไม่ใช่ใน function)
